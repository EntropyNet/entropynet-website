Rules

General network-wide rules

  • No spamming, flooding or advertisement.
  • Do not ask for operator privileges.
  • Do not link to shock sites or questionable pornography.
  • No ASCII art except in #ascii.

Bots

  • If you want to use a bot, it should not join any channels you are not the owner of.
  • If possible, set umode +B on your bot so that opers know easily that it is a bot.

Channel Moderation

  • Network operators are not responsible for moderating your channel.
  • Before asking for help with ChanServ make sure to read /msg ChanServ help thoroughly.
